# 📸 Dossier IMAGES

Mets tes images de projets dans ce dossier.

## Formats acceptés
- `.jpg` (recommandé pour les photos)
- `.png` (recommandé pour les screenshots, logos)
- `.webp` (le plus léger, si tu sais en créer)

## Noms de fichiers
Utilise des noms simples, en minuscules, sans accents ni espaces :
- ✅ `projet-1.jpg`, `aesio-marque-employeur.jpg`, `cpam-event.png`
- ❌ `Mon Projet (final).JPG`, `Projet Été 2024.png`

## Tailles
- Minimum recommandé : 1200px de large
- Idéal : 1600px de large
- Poids max par image : 300 Ko (compresse sur [squoosh.app](https://squoosh.app))

## Comment les utiliser

Dans `projects.js`, à la ligne `image:` :

```
image: "assets/images/projet-1.jpg",
```
